Script editor

SCI Companion includes a script editor which provides a number of improvements over the one in SCI Studio:

  • Automatic parenthesis highlighting as you type
  • Add new words to the vocabulary directly from the script
  • Find text across all script files
  • Better tabbing support (indent blocks of text)
  • Insert object wizards
  • Open header files from code

Opening the editor

To open the script editor, double click on a script, or click the Script editor button in the game explorer's toolbar. From the editor, other scripts can be opened by clicking on them in the Quick links pane. The header files are listed at the top. Header files can also be opened by right clicking on the corresponding include statement in code.
