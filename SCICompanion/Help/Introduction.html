An editor for SCI0 games

SCI Companion will let you edit games you have created with SCI Studio. It provides all of the functionality of SCI Studio including new game creation.

SCI Companion offers these improvements over SCI Studio:

  • More stable operation
  • More advanced, easier to use picture and view editors
  • Option to automatically generate text resources from compiled scripts
  • Smaller code generation for compiled scripts
  • Thumbnail browsers for pictures, fonts, cursors and views
  • Integration of vocabulary file with script editor
  • Better script editor with tab indent support, and parenthesis highlighting
  • Better keyboard support (hot keys that work)
  • ...and much more

SCI Companion was designed for Windows XP. It should also work on Windows 2000, but is known to have problems on Windows 98/ME. Please contact me if you are using Windows 98 or ME, and can't get it to work. For all features to be functional, the GDI+ redistributable must be installed on the latter operating systems (it is included with XP).
