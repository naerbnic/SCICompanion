Insert object wizard

SCI Companion automates adding new object instances to your script.

Adding an object

Click the Insert object button. A dialog comes up that lets you choose the name and class of the object. Supported classes are Prop, Act, Door, Rgn, Locale and Script.

You can optionally add and attach a Script object to most of these classes.

When adding objects such as Views or Acts, the default values for the x, y and view properties are taken from any current fake ego you are using in a pic.
