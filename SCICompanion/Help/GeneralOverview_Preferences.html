Editor preferences

The preferences dialog allows you to manipulate various editor settings. Most of these are explained in the relevant sections. Those that are not, are detailed here.

To reset the preferences to the default (in case the current settings result in a problem that prevents you from accessing the preferences dialog), hold down the SHIFT key while starting SCI Companion.

Drawing tools use grid lines

If this is checked, the drawing tools in the pic editor will have grid lines extending out in each direction. This can be useful for lining up distant objects in a pic.

Use auto-suggest

Use auto-suggest (as opposed to auto-complete) in edit boxes that support it (such as the Add As -> Synonym function of the script editor).

Automatically load last game on open

Open the last game when you start SCI Companion

Views: dupe new cels

This determines whether new cels in a view resource are copied from the previous cel, or start off blank (filled with the transparent colour).

Open some SCI01 games

Though you can't create SCI1 or SCI01 games with SCI Companion, this will let you open and browse the resources of games that have the SCI0 resource.map file format, but SCI1 compression techniques. Quest For Glory 2 is the only game I know of like this.
