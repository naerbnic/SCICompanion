View, font and cursor editors

The view, font and cursor editors share the same layout, and much of the same functionality, so they will be discussed together.

Layout

At the bottom of the screen is an area that lets you choose which cel your are working on. For fonts, this corresponds to the character. For views, this corresponds to a cel in the currently selected loop.

On the left side of the screen is the colour selection pane. It also lists the width and height of the cel you're working on, and contains other features which are specific to the resource type.
