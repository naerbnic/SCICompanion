Kernel Functions¶

Kernel functions are routines built into the interpreter that can be called from scripts. They are essential as they do the graphic drawing, window handling, text printing, file I / O, and everything else you need. The number of kernel functions vary from intepreter versions, but in the interpreter for the SCI0 template game, there are 114. In the SCI1.1 interpreter there are 136.
