SetPort

void SetPort(heapPtr newPort)

Sets the current port to the specified one.

Example

(var hWnd, oldPort)
= oldPort GetPort()

// Draw a white window at 50, 20 with a width of 200 100, a title, and a black foreground.
= hWnd NewWindow(
    50 20 250 120
    "Test Window"
    nwTITLE
    nwON_TOP
    clBLACK
    clWHITE
)

SetPort(hWnd)
Display("Hello in the hWnd window!")
SetPort(oldPort)
Display("Hello in the oldPort!")
DisposeWindow(hWnd)

see also: GetPort
