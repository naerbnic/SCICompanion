Public Procedures

  • number IsPosOrNeg(number aNumber)

    Evaluates aNumber and returns -1 if it is less than zero, 0 if it is equal to zero, or 1 if it is greater than zero.
  • bool EqualsAny(number aNumber, numbers)

    Compares aNumber to all of the numbers given as parameters. If it equals any of them, it returns TRUE, otherwise it returns FALSE.
