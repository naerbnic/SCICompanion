GetFarText

string GetFarText(number rsNumber, number index, string aString)

Reads a string from a text resource and copies it to aString. It loads text resource rsNumber, and accesses the index numbered string. It returns a pointer to aString.

Example

(var aString[40])

GetFarText(100 5 @aString) // reads text entry 5 of TEXT.100 and stores it in aString
