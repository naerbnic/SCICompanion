Include

SCI Studio allows the include keyword to be used to include header files. Header files can contain define keywords, and other include keywords). Include files are very useful for compiling games that use more than one script.

An example header file

/*************************************************************************
 * Example Header
 ************************************************************************/
(include "kernel.sch")
/************************************************************************/
(define rsVIEW    $80)
(define rsPIC     $81)
(define rsSCRIPT  $82)
(define rsTEXT    $83)
(define rsSOUND   $84)
(define rsMEMORY  $85)
(define rsVOCAB   $86)
(define rsFONT    $87)
(define rsCURSOR  $88)
(define rsPATCH   $89)
/************************************************************************/
