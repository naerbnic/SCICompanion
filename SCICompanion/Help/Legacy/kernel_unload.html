UnLoad

void UnLoad(number type, number resNum)

Unloads a resource from memory. If the resource is rsMEMORY, it frees the allocated memory specified by it's pointer.

Example

(var hView, memPtr)
= hView Load(rsVIEW 100) // loads VIEW.100
= memPtr Load(rsMEMORY 1024) // allocates 1024 bytes of memory

UnLoad(rsVIEW 100) // unloads VIEW.000 from memory
UnLoad(rsMEMORY memPtr) // frees memPtr from

see also: Load
