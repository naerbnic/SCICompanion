Class: DCIcon (of DIcon)

The DCIcon script contains an extension of the icon dialog control which allows animation.

Methods

  • void init()

    Creates a new Fwd cycler instance, and sets the cycler property to it. It then initializes the cycler.

  • void dispose()

    If the cycler property is set, it disposes of it. It then disposes of itself.
  • void cycle()

    Cycles and draws if needed the icon object.

  • number lastCel()

    Returns the number of the last cel of the icon object.
