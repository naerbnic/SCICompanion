OnControl

number OnControl(number map, point xy or rect aRect )

Checks to see which controls are at the specified coords or in the rect. It checks the specified map and returns a 16 bit colour map.

The map can be one of the following:

ocVISUAL The visual screen
ocPRIORITY The priority screen
ocSPECIAL The special screen (contains the control information)

There are 16 colours available, and it returns a 16 bit colour map. If a colour is in the region, it's bit is set. So, when checking for a control which is, for example, purple, you wouldn't check for it to be 5, you'd check for it to be 00100000b ($20). For those unfarmilliar with binary, I'll give you a map...

Example

//Using a point
(if(== $1000 OnControl(ocSPECIAL 120 90)
    Print("It is on a red control line!")
)

//Using a rect
(if(== $0200 OnControl(ocSPECIAL 50 20 100 40)
    Print("It is on a blue control line!")
)
