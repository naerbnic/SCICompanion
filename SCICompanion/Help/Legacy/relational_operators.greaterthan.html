Relational Operators: Greater than

If only the > operator is used, it returns the TRUE if code block 1 is greater than code block 2, otherwise, returns FALSE.

If an = operator is appended to the < operator, it returns the TRUE if code block 1 is greater than or equal to code block 2, otherwise, returns FALSE.

If a 'u' is appended to the operator, it treats the two code blocks as unsigned integers.

Syntax

> code block 1 code block 2
>= code block 1 code block 2
>u code block 1 code block 2
>=u code block 1 code block 2
