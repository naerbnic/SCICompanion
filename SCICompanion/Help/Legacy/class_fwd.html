Class: Fwd (of Cycle)

The Fwd class is a general animation cycler for cycling from the beginning to the end of a loop's cels.

Properties

  • client
  • caller
  • cycleDir
  • cycleCnt
  • completed

Methods

  • void doit()

    Increments the client's cel. If the cel is greater than the last cel, the cel is set back to the first cel.
  • void cycleDone()

    Sets the client's cel to the first cel.
