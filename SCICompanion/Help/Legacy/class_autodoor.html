Class: AutoDoor (of Door)

The AutoDoor class is an extension of the Door class which handles all the door related functions. It saves a lot of time when using doors.

Methods

  • void init()

    Sets up the door.
  • void doit()

    If the code property is set, it calls the code's doit() method. If it returns TRUE, the door opens, otherwise, it closes. Otherwise, If the ego is on the door's control, the door opens, otherwise, it closes.
  • void cue()

    Stops update on the door and cues the object pointed to by the notify property.
  • void open()

    Opens the door.
  • void close()

    Closes the door.
