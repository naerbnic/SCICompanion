Inline Assembly

To give you complete control over scripting your game, the ability to to inline assembly was implemented. With inline assembly, you can script SCI byte code within the highlevel code.

Syntax

(asm
    <asm code>
)

Example

(procedure (SomeProc)
  (asm
     pushi #init
     push0
     lofsa {fWindow}
     send 4
  )
)

// the inline assembly is quite versatile. Here's a crafty example:
(if(something)
  (asm jmp _a_goto_label)
)
(asm _a_goto_label: )
