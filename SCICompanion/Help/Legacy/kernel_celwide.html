CelWide

number CelWide(number view, number loop, number cel)

By specifying a the view, loop and cel as a parameter, it looks up the width of the cel and returns it.

Example

(var celWidth, celHeight)
= celWidth CelWide(100 1 3) // Looks up the width of loop 1, cel 3 in VIEW.100
= celHeight CelHigh(100 1 3) // Looks up the width of loop 1, cel 3 in VIEW.100

see also: CelHigh
