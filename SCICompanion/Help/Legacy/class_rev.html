Class: Rev (of Cycle)

The Rev class is a cycler for reverse animation.

Properties

  • client
  • caller
  • cycleDir
  • cycleCnt
  • completed

Methods

  • void doit()

    Sets the client's cel to the next one. If the cel is less than the first cel, the cycleDone() method is called.

  • void cycleDone()

    Sets the client's cel to the last cel in the loop.
